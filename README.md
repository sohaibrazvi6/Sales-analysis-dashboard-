# 📊 Superstore Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard built using the Superstore Sales dataset. The dashboard provides valuable business insights by analyzing sales performance, profitability, customer segments, regional performance, payment modes, and monthly trends.

The objective of this project is to transform raw sales data into meaningful visual insights that support data-driven business decisions.

---

## 🚀 Dashboard Features

- Interactive Region Filter (Central, East, South, West)
- KPI Cards
  - Total Sales
  - Total Profit
  - Total Quantity Sold
  - Average Delivery Time
- Monthly Sales Trend (Year-over-Year)
- Monthly Profit Trend (Year-over-Year)
- Sales by Category
- Sales by Sub-Category
- Sales by Ship Mode
- Sales by Customer Segment
- Sales by Payment Mode
- Regional Sales Distribution

---

## 📈 Key Insights

- Track monthly sales and profit growth across multiple years.
- Compare business performance across different regions.
- Identify the highest-selling product categories and sub-categories.
- Analyze customer purchasing behavior by segment.
- Understand preferred payment methods.
- Monitor operational metrics like average delivery time.
